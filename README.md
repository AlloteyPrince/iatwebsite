# IAT - Innovation in Arts & Tech

A professional portfolio website for IAT, a design agency serving small businesses in Ghana.

## Tech Stack

- Nuxt.js 4 (Nuxt 3+)
- Vue 3
- TypeScript
- Tailwind CSS
- Nuxt Image

## Setup

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Generate static site
npm run generate

# Preview production build
npm run preview
